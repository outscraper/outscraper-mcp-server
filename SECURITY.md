# Security Policy

## Reporting a Vulnerability

Please report suspected security issues by emailing `support@outscraper.com`.

Include:

- a clear description of the issue
- steps to reproduce it
- affected versions or deployment mode
- any proof-of-concept details that help us verify the report

Please do not open public GitHub issues for security-sensitive reports.

We will review the report, confirm impact, and coordinate a fix and disclosure timeline as appropriate.
